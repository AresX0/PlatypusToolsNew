Overview

PlatypusTools is a powerful, all-in-one Windows system utility that combines file management, multimedia editing, system cleanup, security tools, and metadata editing into a single, modern WPF application.

What's New in v3.1.1.1

  • Video Similarity Detection - Find similar videos using perceptual hashing
  • Transition Picker UI - Visual interface for selecting video transitions
  • PDF Encryption/Decryption - Password protect PDFs with permission controls
  • Timeline Drag-Drop - Drag media files directly from File Explorer
  • Metadata Templates - Save and apply metadata templates to files
  • Archive Password/Split - Create password-protected and split archives
  • Recent Workspaces UI - Redesigned with tabs, pinning, and filtering
  • Hierarchical Multimedia Tab - Reorganized into Audio, Image, Video, Media Library, and External Tools sub-tabs
  • Native Video Player - Built-in video player with fullscreen support and popup controls
  • Native Audio Trimmer - Trim audio files with precise start/end time selection
  • Native Image Editor - Rotate, flip, grayscale, invert, and save in multiple formats
  • Fullscreen Popup Controls - Hover at bottom of screen to show controls in fullscreen mode
  • Improved System Audit - Enhanced security scanning with auto-fix capabilities

üìÅ File Management

The File Management tab contains tools for organizing, cleaning, and managing your files.

File Cleaner

Scan and clean folders based on file types, patterns, and naming conventions. This tool combines file scanning with batch renaming capabilities.

How to Use

  1. Browse - Click Browse to select the folder you want to clean
  2. Set Filters - Choose file type filters (e.g., *.tmp, *.log)
  3. Configure Options - Set naming conventions, space/separator conversion
  4. Scan - Click Scan to find matching files
  5. Preview - Review the list of files and proposed changes
  6. Execute - Apply the cleaning/renaming operations

Naming Options

OptionDescription

Duplicates Scanner

Find and remove duplicate files to free up disk space. Supports both exact hash matching and visual similarity detection for images.

Features

  • Hash-based Detection - Find exact duplicates using MD5/SHA256 hashing
  • Similar Images - Find visually similar images using perceptual hashing
  • Staging Area - Stage files for review before deletion
  • Smart Selection - Auto-select newest, oldest, largest, or smallest duplicates
  • Preview - Preview files before deleting

Find and remove empty directories that clutter your file system.

  • Recursive scanning of subdirectories
  • Option to delete or just list empty folders
  • Exclude patterns for system folders

üîß Troubleshooting

Common Issues

FFmpeg/ExifTool not found

  • Ensure FFmpeg and ExifTool are installed in the Tools folder or system PATH
  • Set the correct path via File ‚Üí Set Default Tools Folder
  • Check that ffmpeg.exe, ffprobe.exe, and exiftool.exe are present
  • The app will auto-detect if these tools are in your system PATH

Access Denied errors

  • Run PlatypusTools as Administrator
  • Some system folders require elevated privileges
  • Check file/folder permissions

Video/Audio playback issues

  • Ensure Windows Media codecs are installed
  • Try converting the file to a more common format (MP4, MP3)
  • Check that the file is not corrupted

System Audit Fix buttons not working

  • Run PlatypusTools as Administrator for auto-fix features
  • Some issues cannot be auto-fixed and require manual intervention
  • Check the "CanAutoFix" status - only items with this enabled can be fixed

Fullscreen controls not appearing

  • Move your mouse to the bottom 150 pixels of the screen
  • Controls appear on hover and auto-hide after 2.5 seconds
  • Use keyboard shortcuts as an alternative (Space, Escape, Arrow keys)

Log Files

Logs are saved to the Logs folder in the application directory.

Access via Help ‚Üí View Log or check %APPDATA%\PlatypusTools\Logs\

Getting Help

If you encounter issues not covered here:

  1. Check the log files for error details
  2. Ensure all dependencies are installed (FFmpeg, ExifTool)
  3. Try running as Administrator
  4. Reset configuration to defaults (File ‚Üí Reset Configuration)
  5. Report issues on GitHub: github.com/dxsteel/PlatypusTools
