This folder contains archival copies of all PowerShell scripts and related helper scripts from the historic PlatypusTools PowerShell project.

Purpose:
- Keep original `.ps1` sources together and separate from the new .NET port (`PlatypusTools.Net/`).
- Serve as a preserved reference for future audits and portability.

Notes:
- These are copies; the originals are still present elsewhere in the repository until a later cleanup step (which will be done only with explicit approval).
- Do not modify files here directly; edit the working scripts in the main tree and re-run the archival copy step if you want to refresh the archive.
