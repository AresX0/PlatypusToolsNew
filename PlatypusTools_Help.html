Overview

PlatypusTools is a powerful, all-in-one Windows system utility that combines file management, media conversion, system cleanup, security tools, and metadata editing into a single, easy-to-use application.

Key Features

üìÅ File Cleaner

üßπ System Cleanup

üîí Security Tools

Hide folders, scan for vulnerabilities, manage local users.

üìä Metadata Editor

View and edit file metadata including EXIF, IPTC, and XMP tags.

System Requirements

  • Windows 10/11 (64-bit recommended)
  • PowerShell 5.1 or later
  • Administrator privileges (for some features)
  • FFmpeg/FFprobe (for media operations)
  • ExifTool (for metadata operations)

üìÅ File Cleaner

The File Cleaner tab helps you scan, filter, and clean files from specified directories.

How to Use

  1. Browse - Click Browse to select the folder you want to clean
  2. Set Filters - Choose file type filters (e.g., *.tmp, *.log)
  3. Scan - Click Scan to find matching files
  4. Review - Review the list of files found
  5. Clean - Click Clean to delete selected files

Tip: Use the "Recursive" option to scan subdirectories as well.

üé¨ Media Conversion

The Media Conversion tab provides tools for working with video and image files.

Video Combiner

Merge multiple video files into a single output file.

  1. Add video files using the "Add Files" button
  2. Arrange the order using Up/Down buttons
  3. Set the output filename and location
  4. Select output format (MP4, MKV, AVI, MOV)
  5. Click "Combine" to merge the videos

  • Percentage - Scale by a percentage (e.g., 50% = half size)

üîç Duplicates

Find and remove duplicate files to free up disk space.

Detection Methods

MethodSpeedAccuracyDescription
MD5 HashFastHigh128-bit hash, good for most uses
SHA256 HashSlowerVery High256-bit hash, cryptographically secure
FilenameVery FastLowMatch by name only (may have false positives)
Size + NameFastMediumMatch by size and filename

How to Use

  1. Select folder(s) to scan
  2. Choose comparison method
  3. Click "Scan for Duplicates"
  4. Review groups of duplicate files
  5. Select which copies to delete (keep at least one)
  6. Click "Delete Selected" to remove duplicates
Tip: Always review duplicates before deleting. The tool keeps at least one copy by default.

üîí Security

  2. Click "Hide Folder" to apply hidden + system attributes

üìä Metadata

Supported Metadata Types

üîß Troubleshooting

Common Issues

FFmpeg/ExifTool not found

  • Ensure FFmpeg and ExifTool are installed in the Tools folder
  • Or set the correct path via File ‚Üí Set Default Tools Folder
  • Check that ffmpeg.exe, ffprobe.exe, and exiftool.exe are present

Access Denied errors

  • Run PlatypusTools as Administrator
  • Some system folders require elevated privileges
  • Check file/folder permissions

Video conversion fails

  • Check that input files are valid video files
  • Ensure all videos have the same codec/resolution for combining
  • Check available disk space for output
  • Review the log file for FFmpeg error messages

Metadata editing doesn't save

  • Check that the file is not read-only
  • Some formats have limited metadata support
  • Ensure ExifTool is properly installed

Log Files

Logs are saved to C:\ProgramFiles\PlatypusUtils\Logs\

Check log files for detailed error messages and operation history.

Getting Help

If you encounter issues not covered here:

  1. Check the log files for error details
  2. Ensure all dependencies are installed
  3. Try running as Administrator
  4. Reset configuration to defaults
